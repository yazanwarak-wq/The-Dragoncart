# The-Dragoncart
I will be building a two seater cart and acualy make it work but also leave some space for people to make it themselves and customize. This is sponcerd by Hackclub


Story:
Me and my buddys hav always wanted to drive around the streets and be able to go to one another, becaus we live so far from eachother. Busses are a option
But it is no fun nor is it as relyable and you have to wait for the correct timing.
We have always planed on building something that was able to get us around, therefore this cart.
I focused mostly on having a passenger but also keeping it somewhat small for storage becaus not all of us have that much space or a spare garage.

WARNING!!! This is a work in progress Project
